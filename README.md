# 萤火 前端

> 基于 `React` + `umi` + `antd`

## 开发

### 环境要求

- `Node` 版本 15+
- `Yarn` 最新版本即可

### 本地开发启动命令

```bash
yarn
yarn dev
yarn start // 使用mock数据
```

## 开发一个新页面

1. `/src/pages` 下创建新页面
2. `/config/routes.js` 中添加新路由

## 部署

入口文件为 /dist/index.html

```bash
yarn build
```
